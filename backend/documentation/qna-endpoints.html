Question Endpoints

  1. Submit a Question

    • Request Type: POST
    • URL: http://localhost:3000/questions
    • Body:
      { "question": "What is Node.js?", "category": "60d0fe4f5311236168a109ca", "tags": ["JavaScript", "Backend"] }

  2. Answer a Question

    • Request Type: PUT
    • URL: http://localhost:3000/questions/60d21b4667d0d8992e610c85/answer
    • Body:
      { "answerText": "Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine." }

  3. Get All Questions

    • Request Type: GET
    • URL: http://localhost:3000/questions

  4. Get a Specific Question by ID

    • Request Type: GET
    • URL: http://localhost:3000/questions/60d21b4667d0d8992e610c85

  5. Update a Question

    • Request Type: PUT
    • URL: http://localhost:3000/questions/60d21b4667d0d8992e610c85
    • Body:
      { "question": "What is Node.js and its uses?", "category": "60d0fe4f5311236168a109cb", "tags": ["JavaScript", "Backend", "Runtime"] }

  6. Delete a Question

    • Request Type: DELETE
    • URL: http://localhost:3000/questions/60d21b4667d0d8992e610c85

Answer Endpoints

  1. Get All Answers for a Specific Question

    • Request Type: GET
    • URL: http://localhost:3000/questions/60d21b4667d0d8992e610c85/answers

  2. Update an Answer

    • Request Type: PUT
    • URL: http://localhost:3000/questions/60d21b4667d0d8992e610c85/answers/60d21b9067d0d8992e610c86
    • Body:
      { "answerText": "Node.js is a JavaScript runtime that allows you to run JavaScript on the server-side." }

  3. Delete an Answer

    • Request Type: DELETE
    • URL: http://localhost:3000/questions/60d21b4667d0d8992e610c85/answers/60d21b9067d0d8992e610c86

User-specific Endpoints

  1. Get All Questions Asked by a Specific User

    • Request Type: GET
    • URL: http://localhost:3000/user/60d0fe4f5311236168a109ca/questions

  2. Get All Answers Given by a Specific User

    • Request Type: GET
    • URL: http://localhost:3000/user/60d0fe4f5311236168a109ca/answers

Recommendation Endpoint

  1. Recommend Questions
    • Request Type: GET
    • URL: http://localhost:3000/recommendations

Category Endpoints

  1. Create a New Category

    • Request Type: POST
    • URL: http://localhost:3000/categories
    • Body:
      { "name": "Backend Development", "description": "Questions related to backend development." }

  2. Get All Categories

    • Request Type: GET
    • URL: http://localhost:3000/categories

  3. Update a Category

    • Request Type: PUT
    • URL: http://localhost:3000/categories/60d0fe4f5311236168a109cb
    • Body:
      { "name": "Backend Development and Databases", "description": "Questions related to backend development and database management." }

  4. Delete a Category

    • Request Type: DELETE
    • URL: http://localhost:3000/categories/60d0fe4f5311236168a109cb
