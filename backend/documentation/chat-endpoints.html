API Documentation

1. Get All Chat Options

  • URL: http://localhost:3000/api/chat/
  • Method: GET
  • Query Parameters:
    • page (optional): Page number (default: 1)
    • limit (optional): Number of results per page (default: 5)
    • sortBy (optional): Field to sort by
    • filterBy (optional): JSON string to filter by fields
    • query (optional): Search query

2. Create a New Chat Option

  • URL: http://localhost:3000/api/chat/create
  • Method: POST
  • Body:
    { "title": "string" }
